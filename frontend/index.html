Fine-Tune on RunPod

Evaluate: Base vs Fine-Tuned

Runs both the base model and your fine-tuned model on the held-out evaluation set. Compares using exact match, F1 score, and LLM judge metrics.

Deploy with vLLM

Deploys your fine-tuned model on RunPod using vLLM. Provides an OpenAI-compatible API endpoint with ready-to-use sample code in Python, cURL, and JavaScript.
